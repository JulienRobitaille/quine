spawn :: m a -> m (m a) Source

Start a task, this task may be run out of order with your own context.

After running this command, assume that the context is scrambled.

After running the returned command, also assume that the context is scrambled.

execTask :: MonadIO m => Word64 -> IO () -> Task a -> m a Source

execTask delay idle task will run a task on the GPU and should only be run in the bound thread that is controlling the user interface.

When we block waiting on a sync from the GPU and have no ready fibers then we'll block for up to delay nanoseconds and if that timeout expires, we'll run idle
